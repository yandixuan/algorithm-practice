# 两数之和

```java
class Solution {
    public int[] twoSum(int[] nums, int target) {
        // 只会存在一个有效答案
        // 数组中同一个元素在答案里不能重复出现
        // value-index
        HashMap<Integer, Integer> map = new HashMap<>(16);
        for (int i = 0; i < nums.length; i++) {
            int value = nums[i];
            int left = target - value;
            if (map.containsKey(left)) {
                return new int[]{i, map.get(left)};
            }
            map.put(value, i);
        }
        return new int[]{-1, -1};
    }
}
```

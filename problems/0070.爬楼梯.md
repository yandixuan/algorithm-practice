# 爬楼梯

## 动态规划

```java
class Solution {
    public int climbStairs(int n) {

        // 每次你可以爬 1 或 2 个台阶
        // 爬到n阶楼梯的方案输
        int[] dp = new int[n + 1];
        dp[1] = 1;
        dp[0] = 1;
        for (int i = 2; i < n + 1; i++) {
            dp[i] = dp[i - 2] + dp[i - 1];
        }
        return dp[n];
    }
}

```

## 斐波那契数列通项公式

```java
class Solution {
    public int climbStairs(int n) {
        double sqrt5 = Math.sqrt(5);
        double fibN = Math.pow((1 + sqrt5) / 2, n + 1) - Math.pow((1 - sqrt5) / 2, n + 1);
        return (int) (fibN / sqrt5);
    }
}
```

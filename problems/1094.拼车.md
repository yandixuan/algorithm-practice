# 拼车

## 差分数组

```java
class Solution {
    public boolean carPooling(int[][] trips, int capacity) {
        // 1 <= trips.length <= 1000
        // 差分数组
        int[] diff = new int[10001];
        for (int[] trip : trips) {
            // 上车
            diff[trip[1]] += trip[0];
            if (trip[2] < diff.length) {
                // 下车
                diff[trip[2]] -= trip[0];
            }
        }
        if (diff[0] > capacity) {
            return false;
        }
        // 判断站点是否超载
        for (int i = 1; i < diff.length; i++) {
            diff[i] += diff[i - 1];
            if (diff[i] > capacity) {
                return false;
            }
        }
        return true;
    }
}

```

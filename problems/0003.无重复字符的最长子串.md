# 无重复字符的最长子串

## 滑动窗口

```java
class Solution {
    public int lengthOfLongestSubstring(String s) {
        // 字符-索引位置
        int[] cache = new int[128];
        // 最大不重复子串的长度
        int maxLen = 0;
        // 滑动窗口左指针
        int left = 0;

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);
            // 判断是否包含重复字符
            // 1. abca
            // 2. abba
            // 有可能a的位置小于left，所以每次选择有效的左边界即[left,cache[c]]的最大值
            if (cache[c] != 0) {
                left = Math.max(left, cache[c]);
            }
            // 更新字符的索引位置
            cache[c] = i + 1;
            maxLen = Math.max(i - left + 1, maxLen);
        }
        return maxLen;
    }
}
```
